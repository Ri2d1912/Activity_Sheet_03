Web Programming 1

This course introduces comprehensive introduction to web programming and provides concrete knowledge in web technologies. Students build interactive, secure and powerful web programs while covering client and server side technologies for the World Wide Web. This course explores the technical foundation of the Web and learns key techaologies.

  2. introduction to HTML22
  3. Web Application Development Tools
  4. introduction to Web Programming Language
  5. Using Web Forms
  6. Web Program Database Design and Creation
  7. Web Program and Database Integration
  8. Data Validation and Presentation
